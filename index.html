DogGoSocial

A high-performance social utility for dog owners. Engineered with React Native, Supabase, and PostgreSQL.

Solving Real-World
Pet Isolation.

The core problem: Dog owners want to socialize their pets but lack real-time data on park activity. Browsing static reviews isn't enough.

DogGoSocial functions as a live "status board" for the community. By combining GPS tracking with social check-ins, I created a platform where owners can verify the "vibe" of a park before they even leave their house.

Tactile Check-Ins

The app utilizes expo-av to trigger sound cues during check-in/out. Paired with Haptics.notificationAsync, this provides physical confirmation, ensuring the interface feels responsive even when the device is not in direct line-of-sight.

  Haptics.notificationAsync(type === 'in' ? Success : Warning);
  const { sound } = await Audio.Sound.createAsync(
    { uri: type === 'in' ? IN_URL : OUT_URL }
  );
  await sound.playAsync();

Supabase & PostGIS

Spatial Database Logic

By using **PostGIS Point geography** in Supabase, I enabled native SQL spatial queries. This allows for highly efficient radius filtering (e.g., fetching all active dogs within 500 meters) without overloading the React Native client with unnecessary data processing.
